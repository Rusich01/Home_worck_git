Список команд, которые мы рассматривали на занятии по работе с терминалом

  • Команда "Git status " проверка статуса репозитория
  • Команда"Git add ..." Добавление отдельных файлов
  • Команда " Git commit -m " Комментарий для записи изменений в репозиторий
  • Команда " Git push " выгружает содержимое в удаленный репозиторий
